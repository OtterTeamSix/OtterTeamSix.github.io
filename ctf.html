OtterTeamSix's CTF Resources

CTF Introductions:

  • OverTheWire - Multiple progressive introductions to CTF concepts that build upon previous lessons. Recommended start with Bandit series which deals with priviledge escalation in linux
  • Hacker101 - A website designed to introduce people to hacking concepts. It has a beginner ctf built into it as well as tools and resources to help on your hacker/ctf journey
  • RingZer0 Ctf - Another online persistent CTF that is accessible for beginners up to more advanced users
  • PicoCTF - An online CTF system that offers learning and practice options as well as runs competitions throughout the year
  • MyTechnoTalent's Reverse Engineering training

CTF Tools

Decryption and Crytography Tools

  • CyberChef - A magical drag and drop resource that has everything from rot13 to rsa decryption. Includes Frequency analysis tools as well as more simple hex converters
  • DCode - A comprehensive resource that covers numerous common as well as uncommon crtypographic systems. Gives encoding and decoding options as well as keys
    • RSA Cipher tool
  • JsNice - A great online Javascript (de)obfuscation tool
  • Integer Factorization Calculator - A tool for prime factoring a large number for RSA cracking. (Can take a long time with big numbers maybe build a python tool for this?)
